# AI Personality Project

Проект по созданию антропоморфного ИИ с личностью, эмоциями и характером.

## Ресурсы разработки:

- Google Colab - для экспериментов и тестирования
- Hugging Face Spaces - для демонстрации
- GitHub Codespaces - среда разработки  
- GitHub - контроль версий
- Render - развертывание

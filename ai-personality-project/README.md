# 🧠 AI Personality - Эмоциональный ИИ

Интерактивная демонстрация антропоморфного ИИ с личностью и эмоциональным интеллектом.

## 🎯 Возможности

- **Эмоциональный анализ** текста в реальном времени
- **Динамическая модель настроения** с учетом личности
- **Визуализация эмоционального состояния**
- **Адаптивные реакции** на основе черт характера

## 👤 Персонаж

**Алиса, 28 лет** - ИИ с научным бэкграундом в когнитивной психологии

**Черты личности:**
- Высокая открытость (9/10)
- Умеренная добросовестность (7/10) 
- Низкая экстраверсия (3/10)
- Высокая доброжелательность (8/10)
- Низкий невротизм (4/10)

## 🛠 Технологии

- **Python 3.11** - основной язык
- **PyTorch** - машинное обучение
- **Transformers** - NLP модели
- **Gradio** - веб-интерфейс
- **Hugging Face Spaces** - хостинг

## 🚀 Быстрый старт

1. **Введите текст** в поле ввода
2. **Нажмите "Анализировать эмоции"** или Enter
3. **Наблюдайте** эмоциональную реакцию Алисы
4. **Изучите** детальную разбивку эмоций
